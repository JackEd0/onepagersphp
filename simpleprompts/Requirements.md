# Requirements

## Core Features

-   View a list of prompts from a Notion database.
-   Search and filter prompts.
-   Mark prompts as favorites.
-   Filter prompts by tags.
-   Copy prompt content to the clipboard.
-   Responsive, mobile-first design.

## Technical Requirements

-   **Frontend:**
    -   Framework: Vite + Vue.js
    -   UI Library: Vuetify
    -   Primary Color: `#bd93f9`
-   **Backend:**
    -   Language: PHP
    -   API: A simple script to fetch data from the Notion API.
-   **Data Source:**
    -   Notion Database

## Future Enhancements

-   User authentication.
-   Create, edit, and delete prompts.
-   Offline support.
